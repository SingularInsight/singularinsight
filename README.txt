
# Singular Insight — Data-driven Static Site (No Build Tools)

Centralized content in two files so you update once and it flows across pages:

- `data/company.js` — brand, contact, booking links, global notes
- `data/offerings.js` — packages/services and the engagements table

Pages render from those data files using `scripts/site.js` (vanilla JS).

## Customize
- Edit contact info or booking link in `data/company.js`.
- Edit prices, buttons, or add/remove packages in `data/offerings.js`.
- Replace `assets/logo.png` with your logo file.
